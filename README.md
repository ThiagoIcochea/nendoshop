# NendoShop – Nendoroids E-commerce Platform

## English Version

### Description
NendoShop is an online store dedicated to **Nendoroids**, collectible figures inspired by anime, manga, and popular games. The platform allows fans to browse, purchase, and stay updated with the latest releases.

### Features
- Browse Nendoroid collections by category, series, or character
- View detailed product information
- Secure checkout and order management
- Responsive design for desktop and mobile devices
- Future integration with inventory management and notifications

### Technology Stack
- **Frontend:** React.js
- **Backend:** Node.js + Express
- **Database:** MongoDB Atlas
- **Deployment:** Render (Frontend & Backend) / Optional Railway backend
- **Version Control:** GitHub

### Target Audience
Anime and manga enthusiasts, gamers, and figure collectors aged **15–35** who value official, high-quality Nendoroids and fast, secure online shopping.

## Versión en Español

### Nombre del Proyecto
**NendoShop – Plataforma de E-commerce de Nendoroids**

### Descripción
NendoShop es una tienda en línea dedicada a **Nendoroids**, figuras coleccionables inspiradas en anime, manga y videojuegos populares. La plataforma permite a los fans explorar, comprar y mantenerse al día con los últimos lanzamientos.

- Ver información detallada de cada figura

- Diseño responsivo para desktop y móviles
- Integración futura con gestión de inventario y notificaciones

### Tecnologías
- **Frontend:** React.js
- **Backend:** Node.js + Express
- **Base de datos:** MongoDB Atlas
- **Despliegue:** Render (Frontend y Backend) / Opcional Railway backend

### Público Objetivo
Entusiastas de anime y manga, gamers y coleccionistas de figuras de **15 a 35 años** que valoran Nendoroids oficiales y de alta calidad, y desean compras en línea seguras y rápidas.
